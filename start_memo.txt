## STEP 1. 開発

---

## STEP 2. Git に保存して GitHub と同期

```bash
cd /c/GodotProjects/meirin-battle-godot

git add .
git commit -m "update: <具体的な内容>" 
git push
```

---

## STEP 3. ChatGPT 用 “URL 一括リスト” を生成

```bash
git ls-files '*.gd' '*.json' \
  | sed 's|^|https://raw.githubusercontent.com/HirokiTakah/meirin-godot-game/main/|' \
  > urls_raw.txt
```

---

## STEP 4. ChatGPT に読み込ませる

新しいチャット開始時に、以下を貼る：

---

**テンプレート**（あなた専用）

```
この GitHub リポジトリの最新ソースコードを、web ツールを使って読み込んでください。
https://github.com/HirokiTakah/meirin-godot-game

また、urls_raw.txt にあるすべての raw リンクを読み込んでください。
HTML ではなく raw.githubusercontent.com の生ファイルを使ってください。
```

---

## STEP 5. ChatGPT が解析 → 設計 or コード生成

あなたは次のように依頼すればよい：

* 「上流・中流と矛盾がないか検証して」
* 「RPG 型バトル計算式を提案して」
* 「EventScene の進行を実装して」
* 「stages.json の ch01 を自動生成して」

ChatGPT は **最新のJSON・GDScript** をもとに回答します。
